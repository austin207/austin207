## Active Work

<table>
<tr>
<td width="50%" valign="top">

**Virtus — Autonomous Porter Robot**
Differential-drive robot for airport luggage handling. Running ROS 2 Jazzy on RPi 5 (16GB) with an RPi AI HAT+ (Hailo-10H, 40 TOPS) for edge inference. Hardware loop: RPi → ESP32 motor controller → dual BTS7960 H-bridges → 40Kgcm DC motors. ESP32 sensor fusion node runs a Kalman filter. Business model: B2B airport operators.

</td>
<td width="50%" valign="top">

**M.Tech Thesis — PCM Simulation**
Electrothermal simulation of mushroom-type and confined/pillar-type Phase Change Memory cells in COMSOL Multiphysics (Heat Transfer in Solids + Electric Currents, 2D axisymmetric). GST conductivity modeled with Poole-Frenkel + Arrhenius. Stationary study complete. Time-dependent pulse study and HfO2 interfacial layer in progress.

</td>
</tr>
