<h2 align="center">Active Work</h2>

<div align="center">
<table>
<tr>
<td width="50%" valign="top">
<h3 align="center">Virtus — Autonomous Porter Robot</h3>
<p align="center">
<img src="https://img.shields.io/badge/ROS_2_Jazzy-Active-64FFDA?style=flat-square&labelColor=112240"/>
<img src="https://img.shields.io/badge/Hailo--10H-40_TOPS-64FFDA?style=flat-square&labelColor=112240"/>
</p>

Airport luggage handling robot on ROS 2 Jazzy. Raspberry Pi 5 (16GB) + Hailo-10H AI HAT+ for 40-TOPS edge inference. Drive chain: RPi to ESP32 motor controller to dual BTS7960 H-bridges to 40Kgcm DC motors. ESP32 sensor fusion node runs a Kalman filter. B2B target: airport operators at Rs.7L/unit + Rs.1.1L ARR.

</td>
<td width="50%" valign="top">
<h3 align="center">M.Tech Thesis — PCM Simulation</h3>
<p align="center">
<img src="https://img.shields.io/badge/COMSOL-Active-64FFDA?style=flat-square&labelColor=112240"/>
<img src="https://img.shields.io/badge/Stationary_Study-Complete-64FFDA?style=flat-square&labelColor=112240"/>
</p>

Electrothermal simulation of mushroom-type and confined/pillar-type Phase Change Memory cells. 2D axisymmetric model using Heat Transfer in Solids + Electric Currents physics. GST conductivity modeled with Poole-Frenkel + Arrhenius (beta=3.509e-24, Ea=0.09eV). HfO2 layer and time-dependent pulse study in progress.

</td>
</tr>
<tr>
<td width="50%" valign="top">
